Facebook Emoji Keyboard Shortcuts available in Messenger Chat

All 30 keyboard shortcuts for v9 of emojis

Facebook changed its emoji set a lot of times. According to their naming scheme, the current version is the 9th iteration of emoji sets.

This change also came with a change of the keyboard shortcuts used to create the emojis in its chat.

Currently there is no official list of Messenger emoji keyboard shortcuts.

So I decided to make one by sending 730.000 key combinations to Facebook, and save the ones who returned emojis.
